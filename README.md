# Logic Circuits - Pratical Exercices

This directory has as objective a more efficient management, organization and storage for pratical exercices proposed in our "Logic Circuits" classes on Universidade Federal do Rio de Janeiro (UFRJ). This exercices are used as evaluation method to test our understanding of the subject.

## organization

- Each subdirectory had in their own **README** the teacher intructions for the practice covered. The major part of each pratice doesn't involve code, so this repository will be used mainly for organization and management of the pre and post lab report and other needed documents.

- Each commit will adopt the **conventional commit**
